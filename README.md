# Home de la Organización EDA 1 / Prog 2 - UNLu

Este repositorio contiene la **página principal** de GitHub Pages para la organización `unlu-prog-2`.

Se genera automáticamente a partir de los archivos Markdown usando Jekyll.

## Estructura del sitio

- `_config.yml` - Configuración de Jekyll
- `index.md` - Página principal con información completa de las materias
- `guia-eda1.md` - Landing específico para EDA 1 (plan nuevo)
- `guia-prog2.md` - Landing específico para Programación 2 (plan viejo)
- `imagenes/` - Imágenes del sitio
- `README.md` - Este archivo

## 🌐 Ver el sitio

Visita: **https://unlu-prog-2.github.io**

## ⚙️ Desarrollo

```bash
cd unlu-prog-2.github.io

# Editar archivos .md
# Hacer commit y push para actualizar el sitio
git add -A

git push
```
